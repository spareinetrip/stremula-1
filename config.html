Configure Reddit API

This addon requires Reddit API credentials to fetch Formula 1 posts. Without these, the addon cannot work.

üîß How to Get Reddit API Credentials

  1. Go to Reddit App Preferences
  2. Click "Create App" or "Create Another App"
  3. Choose "script" as the app type
  4. Enter any name (e.g., "Stremula1")\li>
  5. Copy the client ID (under the app name) and secret
  6. Use your Reddit username and password

Note: These credentials are stored as environment variables and are required for the addon to function.
